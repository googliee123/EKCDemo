Block-Level Elements

  • <div>: Used for grouping content.
  • <section>: Represents a section of content.
  • <article>: Represents an independent piece of content.
  • <header>: Represents the header of a section.
  • <footer>: Represents the footer of a section.
  • <main>: Represents the main content of the document.
  • <nav>: Used for navigation links.
  • <aside>: Represents content related to the surrounding content.
  • <form>: Defines an HTML form for user input.
  • <p>: Represents a paragraph of text.

Inline Elements

  • <span>: A generic container for inline content.
  • <a>: Defines a hyperlink.
  • <strong>: Defines bold text (strong emphasis).
  • <em>: Defines italic text (emphasis).
  • <br>: Inserts a line break.
  • <img>: Embeds an image.
  • <code>: Represents computer code.
  • <q>: Defines a short inline quotation.
  • <mark>: Highlights text (used for marking important text).
  • <abbr>: Defines an abbreviation or acronym.
